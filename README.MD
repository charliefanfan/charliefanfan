 我是查理,現任遊戲公司的大數據產品經理。
 希望能用算法跟數據來幫助遊戲的各個環節！
 
#### 遊戲行業的數據能做什麽
從數據中挖掘數據價值，對於不斷進行優化遊戲活動設計、場景設計等，有很大的幫助。
除了基礎運營數據分析支持外，還能進行遊戲禮包分析、用戶流失分析、市場投放素材分析等各種方面各維度分析。
如果有算法工程師的配合可以進行ltv預測,(亞)流失預測等預測,或是加上特徵與模型進行遊戲內的各種個性化推薦。
